Design a modern, minimalist landing page for an electric vehicle (EV) brand, built around a 3D car model with scroll-driven animation. Style: sleek, futuristic, premium — inspired by Tesla, Lucid Motors, and Rivian websites.

GLOBAL STYLE
- Color palette: deep charcoal/black background (#0A0A0B), off-white text (#F5F5F5), single electric-blue or neon-green accent color for CTAs and highlights
- Typography: bold, geometric sans-serif for headlines (like Inter or Neue Montreal), light-weight sans-serif for body text
- Generous whitespace, full-bleed sections, no clutter
- Smooth, cinematic scroll transitions between sections (fade + slight parallax)

SECTION 1 — HERO
- Full-viewport height
- 3D EV car model centered, rotating slowly or responding to scroll (car rotates/moves as user scrolls down)
- Large bold headline (e.g. "THE FUTURE, DRIVEN.") with subtext below
- Minimal top navbar: logo left, nav links (Design, Performance, Interior, Charging, Configure) center/right, "Order Now" button right
- Scroll-down indicator icon at bottom

SECTION 2 — DESIGN / EXTERIOR
- 3D model shifts angle as user scrolls (e.g. front view → side profile → rear view)
- Text callouts that fade in beside the model, pointing to design details (aero lines, LED lighting signature, wheel design)
- Sticky/pinned car model on one side, scrolling text panels on the other

SECTION 3 — PERFORMANCE STATS
- Full-width dark section with large animated numbers (0-60 time, range in miles, top speed, horsepower)
- Numbers count up as section scrolls into view
- Subtle grid or line background pattern

SECTION 4 — INTERIOR / TECH
- Horizontal scroll gallery or fade-through image carousel of interior shots
- Highlight touchscreen, seating, sound system with minimal icon + label callouts

SECTION 5 — CHARGING / SUSTAINABILITY
- Split layout: left side text about charging speed & range, right side illustration/animation of charging port or battery cutaway
- Progress-bar style animation showing charge percentage filling as user scrolls

SECTION 6 — CONFIGURATOR / CTA
- Color swatch selector that changes the 3D car's color in real time
- Wheel style selector below
- Large "Build Yours" CTA button

FOOTER
- Minimal dark footer: logo, quick links, social icons, newsletter signup, copyright

INTERACTIONS
- Scroll-triggered fade-ins and slide-ups for all text blocks
- 3D model should feel "pinned" during key sections while content scrolls past it
- Smooth easing on all transitions, no abrupt cuts
- Mobile: stack sections vertically, simplify 3D interactions to auto-rotate instead of scroll-linked
